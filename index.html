About Me

I'm a junior full stack web developer with experience in Ruby on Rails and JavaScript-based programming located in New York City. While helping build my family’s construction business, I discovered web development when I took a basic web design class in college. I have been programming ever since. While learning about web design in college I realized I needed to learn back-end development. So I decided to join Flatiron School.

Flatiron School's full stack web development program was an intense learning experience. Learning to code (or rather, become a coder) involved doing a lot at once--problem-solving, thinking in steps, being concise, translating thoughts into code, and a sort of obsessive curiosity. It was difficult, but well worth it. I recently graduated and am excited to bring my newly sharpened skills to an awesome company.

Sinatra Soccer Training Tracker

I used custom CSS for the front-end, in which I chose to implement a minimal design with neat typography. This was my first time building a full-scale app so I learned a lot about planning, app life cycles, and testing.

Rails Trail Master

Trail Master allows users to share information about dangerous hiking trails. I built it with Ruby on Rails with RESTful routes and then later injected a jQuery front-end. I opted to roll my own authentication logic and I implemented OmniAuth with Facebook. I spent an equal amount of time on the front-end and back-end, using custom CSS for the styling. I learned a lot about translating JSON responses into JavaScript Model Objects.

Technologies: Ruby on Rails, ActiveRecord, SQL, CSS, Bootstrap, HTML, OmniAuth, jQuery, JavaScript.

React Planet Express

React Planet Express is a CRUD app that allows you to share information about outer space travel destinations. The back-end was built with Ruby on Rails with a SQL database and a React/Redux for the front-end. I made use of React Router, Rails API to handle data persistence and retrieval, and Redux middleware to modify state change. The front-end features custom CSS for the styling and Bootstrap for a responsive layout.

Technologies: Ruby on Rails, SQL, CSS, HTML, JavaScript, Bootstrap, React, Redux.
